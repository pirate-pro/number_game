# 数忆数实时对战云开发配置

## 1. 开启云开发

1. 在微信开发者工具中打开本项目。
2. 点击顶部 `云开发`，开通环境（测试环境即可）。
3. 确保项目已经勾选当前云环境。

## 2. 创建集合

需要创建两个集合：

- `players`
- `rooms`

## 3. 测试期权限（开发调试）

为了先跑通联机流程，测试阶段可以把这两个集合权限设置为：

- 所有用户可读可写

上线前请改成更严格的安全规则，并把写入逻辑迁移到云函数。

## 4. 联机流程

- 发起方在 `好友对战` 创建房间并分享邀请。
- 受邀好友打开邀请卡片后，会自动加入同一房间。
- 双方进入同题实时对战，进度通过 `rooms` 文档实时同步。
